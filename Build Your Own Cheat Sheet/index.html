CSS Properties Reference

Common Features

Attribute Description
color Sets the color of text inside an element.
background-color Sets the background color of an element.
font-family Sets the font family for text inside an element.
padding Sets the space between the content of an element and its border.
margin Sets the space between the border of an element and its surrounding elements.
position Sets the positioning behavior of an element.
display Sets the display behavior of an element.
top, right, bottom, left Sets the distance between an element and the edges of its container.
